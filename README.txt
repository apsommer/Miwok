7/5/18 This is an Android learning app focusing on the endangered Native American language Miwok. The app successfully demonstrates the use of multiple activities, explicit intents, custom classes, custom adapters to populate an AdapterView (ListView), MediaPlayer with a completion listener, AudioManager with an audio focus change listener, and XML layouts.

6/9/18 This project was created on a Sony E5823 running Android 7.1.1 (API 25).
